about me

Hey, I'm Tanya - a full stack developer from Atlanta. I'm high-spirited, decisive, and insatiably curious. A serial hobbyist, I absolutlely love learning and trying new things. Being someone that loves to be creative, take things apart and put them back together just to see how they work, and reading about the newest tech and science innovations, I was practically meant to be a developer!

things i've built

Below are a few of the projects I have built, along with links to deployed demos and the Github repositories, which contain the full code and files, more detail about the functionalilty of the apps, and code explanations.

LIRI Node App
Search app built using Node.js. Takes in user input as a command and a search query. Responds with approriate results from Spotify, BandsInTown, or OpenMovieDatabase. Uses several node packages for user input and styling.
Demo is a GIF example app running in terminal as it cannot be deployed.

demo - gh repo

Super Smash Babies
RPG Style Game built with jQuery and JavaScript. Player chooses their character and opponent. Every time the 'attack' button is clicked, the characters stats are updated in real time. Game ends when the player is defeated or the player defeats four opponents.

demo - gh repo

World Geography Trivia Game
Trivia Game using JavaScript, demonstrating use of timeouts. Player starts game, random question is asked. Player has 10 seconds to make a guess. Message appears whether answer was correct, incorrect, or if time was up. After 10 questions, players game stats are shown, along with a button to play again.

demo - gh repo

More Than Words
App utilizing GIPHY's API, demonstrating use of RESTful API's and ajax functions. User can select one of the presuggested tags to search, or create their own tag by searching. Searches are saved as a button - option exists to clear searches.

demo - gh repo

SQL Storefront
Command Line storefront using SQL database to store products inventory data, run with Node. Allows user to see products available, price, and quantity. Database updated after each purchase.

demo - gh repo

Squadron - Meet Up App
Group developed app allowing entry of two addresses - returns middle point between places, and nearby businesses. Built using Yelp and Mapquest APIs, Materialize, Leaflet, and jQuery.

demo - gh repo

Guess the Composer - Hangman Game
